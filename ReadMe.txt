------------------setup.py安装方法---------------------
0.下载setuptools：
http://www.lfd.uci.edu/~gohlke/pythonlibs/#setuptools
------------------------------------------------------
1.若为Windows环境，则打开setup.py文件
  在代码开头添加如下几句： 
#-----------------------------------------------------
import sys
reload(sys)
sys.setdefaultencoding('gbk')	# 也可能是'utf8'
#-----------------------------------------------------
2.进入CMD
3.输入DOS命令进入setup.py文件所在目录
4.输入python setup.py install
